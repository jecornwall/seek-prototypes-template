# Brief

Drop your input material for this prototype here:

- PRD (`PRD.md` or `PRD.pdf`)
- Screenshots (`*.png`, `*.jpg`)
- Sketches (`*.pdf`)
- Voice-memo transcripts (`*.txt`, `*.md`)
- Competitor references, prior designs, anything that grounds the agent's work

The agent reads everything in this folder before starting any new screen or feature
(see `CLAUDE.md` and `.cursor/rules`). Treat this folder as the source of truth
for product intent.

Files in this folder are committed to the prototype branch, so anyone looking at
the prototype on GitHub sees the same context you started from.

Do not put production data here. Mock data only. (See `.gitignore`.)
